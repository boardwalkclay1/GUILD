Accessing Options

The gateway to the secret world of options trading begins with unlocking your broker.

The Universal Pattern

Every broker — no matter the name — follows the same pattern:

1. Create an account

2. Verify your identity

3. Enable investing

4. Request options approval

5. Fill out the options questionnaire

6. Wait for approval

7. Options unlock

This page teaches you EXACTLY how to do it on four major brokers — but remember: there are many brokers, and the process is always nearly identical.

Chase (J.P. Morgan)

1. Download Chase or visit chase.com

2. Create an account → enter SSN, address, phone

3. Open a “Self‑Directed Investing” account

4. Go to “Account Management” → “Trading Permissions”

5. Select “Options Trading”

6. Fill out the approval form (experience, income, risk)

7. Submit → approval usually same day

Webull

1. Download Webull

2. Create account → verify identity

3. Tap “Menu” → “Options Trading”

4. Fill out the options questionnaire

5. Select Level 2 (buying calls/puts)

6. Submit → approval usually instant

Robinhood

1. Download Robinhood

2. Create account → verify identity

3. Tap “Investing” → “Enable Options”

4. Fill out the options form

5. Submit → approval within minutes

TD Ameritrade

1. Visit tdameritrade.com

2. Create account → verify identity

3. Go to “Client Services” → “My Profile”

4. Under “General” → request options approval

5. Fill out the form

6. Approval usually 24 hours
